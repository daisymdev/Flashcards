20 MySQL Flashcards

Question Answer
What is MySQL? MySQL stans for "My Structured Query Language." It is a relational database management system
What is the default port for MySQL Server? The default port is 3306.
What are the drivers in MySQL? There are many drivers, including JDBC Driver, ODBC Driver, etc.
What is a Primary Key? A primary key identifies each record in the table uniquely. Each table can only have one primary key.
What is a Foreign Key? A foreign key is what references another table with the primary key. Foreign keys serve as a link between tables.
Is MySQL query case sensitive? No.
How do you create a database in SQL? The syntax would go as follows:
CREATE DATABASE name;
How do you select all columns from a table named "tests"? The syntax would go as follows:
SELECT * FROM tests;
What is the most common join? Inner Join
What SQL statement would you use to select values within a certain range? The statement would be:
BETWEEN
What is the difference between SQL & MySQL
SQL MySQL
SQL is a Structured Query Language, it is used for accessing & managing databases. MYSQL is a relational database management system. A DBMS allows user to interact with database.
What are the 4 joins in SQL?
  • Inner Joins
  • Right Joins
  • Left Joins
  • Full Joins
What are some common SQL Functions? Some common functions are:
CURRDATE(), CONCAT(A, B), FORMAT(X, D), etc.
What are the 3 operators in SQL?
  • Arithmetic Operators
  • Logical Operators
  • Comparison Operators
What are the different relationships?
  • One to One Relationship
  • One to Many Relationship
  • Many to One Relationship
  • Self-Referencing Relationship
What is View? A View is a database object that acts as a virtual table containing limited data you queried for.
What is an Index? An Index is not a table or view in which you can interact with, but it is useful when searching for a WHERE. It increases the speed of querying a database.
What is a Stored Procedure? A Stored Procedure is a stored set of statements that we can reuse later to query.
What are the benefits of using a Function in SQL? The benefit of using Functions is that we can organize, and operate on the data.
What kind of statement will select all records from table named "keyboards" where the value of the column "Name" starts with "h"? The statement would look like this:
SELECT * FROM keyboards WHERE Name LIKE 'h%';
